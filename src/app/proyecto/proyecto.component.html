DESCRIPCIÓN DEL PROYECTO


El actual proyecto tiene como objetivo evaluar los conocimientos aprendidos en la clase de desarrollo web. De igual manera llevar a cabo un proyecto real que está relacionado con el mundo músical y que emplea varios de los conceptos aprendidos en la clase.

REQUERIMIENTOS

Historias de Usuario

Usuario votante


HUV.1. Creación de cuenta
HUV.2. Activación de cuenta
HUV.3. Autenticación de usuario
HUV.4. Ver listas por género musical creadas
HUV.5. Buscar canción una canción
HUV.6. Votar/Desvotar por una lista

Usuario administrador


HUV.1. Autenticación de usuario
HUV.2. CRUD Géneros músicales
HUV.4. CRUD Canciones

ENTREGA 1

Diccionario de datos

Tabla: UsuarioAdmon

Nombre de la columna Tipo de dato Descripción
ID Clave primaria Identificador único del usuario
Nombre Varchar El nombre del usuario administrador
Correo Varchar La dirección de correo electrónico del usuario administrador.
Password Varchar La contraseña del usuario administrador

Tabla: UsuarioVotante

Nombre de la columna Tipo de dato Descripción
ID Clave primaria Identificador único del usuario
Nombre Varchar El nombre del usuario votante
Correo Varchar La dirección de correo electrónico del usuario votante.
Password Varchar La contraseña del usuario votante

Tabla: Listas

Nombre de la columna Tipo de dato Descripción
ID Clave primaria Identificador único de la lista de canciones
genero Varchar El género musical asociado a la lista de canciones
UsuarioAdmonID Clave Foránea El ID del usuario administrador que creó la lista de canciones
UsuarioVontanteID Clave Foránea El ID del usuario votante que creó la lista de canciones

Tabla: Cancion

Nombre de la columna Tipo de dato Descripción
ID Clave primaria Identificador único de la canción
nombre Varchar El nombre de la canción
genero Varchar El género musical al que pertenece la canción
rating INT El rating o calificación de la canción
artista Varchar El nombre del artista intérprete o grupo
album Varchar El nombre del álbum al que pertenece la canción
listasID Clave Foránea El ID de la lista de canciones a la que pertenece esta canción
